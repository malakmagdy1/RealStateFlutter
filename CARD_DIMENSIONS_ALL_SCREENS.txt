==================== KEY NUMBERS ====================

CURRENT STANDARD:
  • maxCrossAxisExtent: 300px (most screens)
  • childAspectRatio: 0.85 (most screens)
  • Calculated Height: 353px
  • Spacing: 10px (most screens)

EXCEPTIONS:
  • Home: 320px width (fixed, horizontal scroll)
  • Company Detail: 0.90 aspect ratio, 24px spacing
  • AI Chat: 380px max width

==================== HOW TO CALCULATE ====================

Given aspectRatio = 0.85:
  • Height = Width ÷ 0.85
  • Example: 300px width → 353px height

To change:
  • Wider cards: Increase maxCrossAxisExtent
  • Narrower cards: Decrease maxCrossAxisExtent
  • Taller cards: Decrease childAspectRatio (e.g., 0.75)
  • Shorter cards: Increase childAspectRatio (e.g., 0.95)
